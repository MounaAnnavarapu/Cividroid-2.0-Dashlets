Final Report On the Cividroid 2.0 - GSoC 2016 project

I am Mouna Annavarapu, a tech enthusiast. I’m from Vijayawada, India and have completed Bachelor’s in Computer Science Engineering at Hindustan University, Chennai. I got first introduced to Cividroid through my mentor, Jagatheesan Pillai. Cividroid, is an Android Application which extends CiviCRM with a mobile perspective, leveraging some of the unique advantages of the Android platform.

While I was so puzzled thinking about how to get myself into a task which could keep me occupied, using the skills I have and also benefit a huge group of people, I found Cividroid quite interesting when I looked into the updates on the app development. I eventually came up with an idea to extend Cividroid’s functionalities. And I’m so glad I made it into GSoC 2016.

Dashlets is destined to merge with Cividroid as Cividroid 2.0. Here is a brief on Dashlets as an independent App: The App revolves around the concept of dashlets, providing users all the CiviReports by syncing functionality which can further be used to create charts displaying the data visually. The CiviReports data can be saved for offline references. The charts can be used to create widgets for the home screens.

Features:

  • The App needs only a apikey, sitekey and url of your website. No login is required to use the App.

  • All the CiviReports (even a custom one, or provided by 3rd party extensions) are synced and are available as a Dashlet.

  • A Dashlet tabulates the data on a page which is scrollable both vertically as well as horizontally. Each page can display upto 25 records. A pair of floating navigational arrow buttons can be used to load more data.

  • Each page of a dashlet can be saved for offline viewing. Switching between online and offline mode is seamless with a button in the toolbar.

  • For every pair of valid columns (with numeric data), a corresponding chart can be generated. App supports the following charts:

    • Bar Chart
    • Horizontal Chart
    • Pie Chart
    • Cubic Line Chart

  • Each chart generated can be saved to stick as a widget on homescreen.

Software Support used for Cividroid 2.0

While I worked on the Dashlets concept for the Cividroid 2.0, different ideas and confusions sparked in my mind for generating the visual charts from the CiviReports. I finally found MPAndroidChart library to render the charts elegantly.

Screens

Few screens for a short glimpse of the final product.

CiviReports are displayed in a list, where a user can select a CiviReport and a detailed spread of its content is displayed. Clicking the floating ‘+’ button will let a user to generate a chart with desired fields. Y-Axis must be a column with numeric values and the App will let the user select only valid columns, while X-Axis could be any column.

These are the pretty looking charts a user can generate using the data in the CiviReports.

Offline Mode

It’s a pretty cool feature I added thinking of the state of the data in the app when it’s not connected to the internet. An icon on the top lets the users save a page for offline viewing. One can always go online using the globe icon while the saved page is open.

Save Widgets

The charts generated can be used as widgets on home screens. The user could save a chart to use as a widget, by selecting “Save for widget” from options and naming it. In the widget set up, selecting the Dashlets widget will let the user choose a saved chart for using as a widget.
